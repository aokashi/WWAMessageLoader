WWAMessageLoader
===

## これは何？
WWAの各パーツのメッセージを表示するアプリケーションです。

ブラウザの検索機能を利用すればメッセージを探すことが可能です。

## 利用方法
1. 表示したいWWAのマップデータを配置します。
2. `index.html` にアクセスします。
3. URLに `?(マップデータ名)` を添えてアクセスします。
4. 結果が表示されます。
